# Grocery Delivery Web App

## Tech Stack

React, TypeScript, Tailwind CSS, Zustand, React Router

## Features

- Splash and onboarding flow
- Login and signup
- OTP verification
- Location selection
- Home page
- Product listing
- Product details
- Search with debounce
- Filters
- Cart
- Favorites
- Checkout
- Order success and failure screens
- Responsive desktop layout

## Run Locally

npm install

npm run dev

## Build

npm run build

## Deployment

Will be available in future on Vercel / Netlify
